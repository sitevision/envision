Envision by Sitevision

Create Sitevision like components and modules the easy way with Envision.

At Sitevision, we believe in making the complex simple. That’s why we created Envision — an open-source, easy-to-use CSS and component framework that streamlines your work with HTML, CSS, and JavaScript.

  • Colors

    Color classes and variables for elements, components and layouts.

  • Typography

    Classes, variables and utils for website text styles.

  • Forms

    Form elements, components, buttons and more.

  • Components

    Ready to use components for your Sitevision projects.

  • Utils

    Miscellaneous utility classes for accessibility, styling and more.

  • Layout

    Utility classes for creating component layouts.

  • Dashboard Widgets

    An extension to Envision that should be used when creating Sitevision Dashboard Widgets.

How to get started

Envision is included in all Sitevision websites. The components are ready to use in your projects.

Usage examples in Sitevision:

  • Site themes
  • Themes in layouts
  • Theme colors in layouts
  • Site fonts
  • CSS rules
  • Custom templates
  • HTML- or Script modules
  • Webapps and Widgets
  • Anywhere where custom HTML, CSS, and JavaScript are used

Contribute

Feel free to check out the Envision GitHub repository. Pull requests and issues are welcome.

To run locally

  1. [Clone repository]
  2. Run yarn install to install dependencies
  3. Run yarn dev to start locally

To build

Run yarn build
